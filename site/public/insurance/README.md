# Insurance logos (optional)

Put your **Cigna logo** here so the Insurance page uses it instead of an external image.

- **Filename:** `cigna.png` (or `cigna.webp` / `cigna.jpg` — if you use a different extension, say so and we’ll update the code)
- **Path:** `site/public/insurance/cigna.png`

After you add the file, the site will show it in the “Insurance plans we accept” section. If you named it something other than `cigna.png`, tell the assistant and they’ll update the path in the code.
