# AZUM Medical Website Rebuild

## Purpose
Rebuild and modernize the AZUM Medical website to improve design, clarity, mobile experience, and appointment conversions.

Current site:
https://azummedical.com/

## Goals
- Modern, professional design
- Mobile-first experience
- Higher appointment conversion
- Clear explanation of services
- Warm concierge-style healthcare brand

## Brand Direction
Hybrid primary care + wellness-focused concierge care.

The clinic experience should feel:
- welcoming
- calm
- personal
- premium but approachable

## Design Influence
Primary brand colors:
Purple and gold from the AZUM logo.

Design aesthetic:
- airy
- minimal
- high-resolution imagery
- subtle gradients
- modern typography
- fewer boxy containers

## Tech Stack
- Next.js
- Tailwind CSS
- Vercel deployment
- GitHub repository

## Conversion Goal
Increase appointment bookings.
