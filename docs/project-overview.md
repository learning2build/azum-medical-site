# Project Overview

## Objective
Modernize the AZUM Medical website while preserving the clinic's welcoming and personalized atmosphere.

## Key Improvements
- Modern design
- Better mobile experience
- Improved appointment conversion
- Clear service explanations
- Stronger trust signals

## Target Patients
Adults seeking:
- primary care
- preventive health
- wellness-focused medical support
- personalized doctor relationships

## Brand Personality
- calm
- trustworthy
- welcoming
- modern healthcare
- boutique clinic feel

## Constraints
We will NOT build:
- custom patient portal
- medical records system
- scheduling system

Booking will link to the clinic’s existing appointment system.

## Success Criteria
A website that looks modern, feels welcoming, and encourages patients to book appointments.
